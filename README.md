# shockwaveads.com — Shockwave Ads LLC marketing site

Live at **https://shockwaveads.com** via GitHub Pages (this repo, `main` branch, custom domain + enforced HTTPS).

**What this is:** a 6-page static site — plain HTML/CSS/JS, no frameworks, no build step. Edit a file, push, live in ~60 seconds.

| File | URL | What it is |
|---|---|---|
| `index.html` | `/` | Landing page: offer, ROI calculator (vanilla JS), embedded GoHighLevel interest form, booking link |
| `free-trial.html` | `/free-trial.html` | How the free 30 days works |
| `faq.html` | `/faq.html` | FAQ |
| `meta-setup.html` | `/meta-setup.html` | Client guide: set up a Meta ad account + Leadsie connect |
| `privacy.html` | `/privacy.html` | Privacy policy (includes A2P SMS consent language) |
| `terms.html` | `/terms.html` | Terms of service |

**House rules for any contributor (human or AI):**
1. All performance numbers are ESTIMATES — never phrase anything as a guarantee (legal/compliance posture).
2. Keep the Privacy/Terms footer links on every page.
3. No secrets or API keys in this repo — it is public.
4. The embedded form/calendar iframes point at the company's GoHighLevel CRM — don't change those IDs without checking the owner's records.
5. Master copies + full operations map live in the owner's private records ("Claude HQ / Shockwave Ad Agency / blueprints/website.md").
